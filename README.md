
# bluewhale

The [Dockerboard][] Web Client.


## Quickstart

### Prerequisites

* [Node.js][]
* [Bower][]
* [Gulp][]

```
npm install -g gulp
npm install -g bower
bower install
npm install
```

### Development

```
gulp watch
```

### Production

```
gulp clean
gulp build
```


[Dockerboard]: https://github.com/dockerboard/dockerboard
[Node.js]: https://nodejs.org
[Gulp]: http://gulpjs.com
[Bower]: http://bower.io
